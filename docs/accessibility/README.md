# Accessibility & inclusive reading

This documentation site is built with accessibility in mind.

---

## Keyboard & navigation

- **Skip to content** — Keyboard-visible skip link available on all pages
- **Search** — Press `/` or `Ctrl+K` to open search; `Esc` closes it
- **Tab through sidebar** — All navigation items are keyboard accessible
- **Copy button** — Code blocks include a copy-to-clipboard button

---

## Visual design & contrast

- **Light (default)** and **dark (slate)** palettes available
- Links are underlined for clarity
- Focus styles are augmented for keyboard navigation

---

## Motion

The site respects `prefers-reduced-motion` — animations are disabled when the user has requested reduced motion.

---

## Mermaid diagrams

Flowcharts and sequence diagrams include a one- or two-sentence summary before or after the diagram block. If you use a screen reader, the alt text provides the same information.

---

## Reporting issues

If you find an accessibility issue, please open a [GitHub issue](https://github.com/anstwechy/mitf_wallet_public_docs/issues/new) with the **accessibility** label.
